File Format Converter

Python 製のファイル形式変換デスクトップアプリ。ドラッグ＆ドロップ対応のモダンな GUI で、画像・PDF・テキストを手軽に変換できます。

⬡ GitHub で見る ↓ ZIP でダウンロード

対応している変換

🖼

画像形式の変換

JPG ⇄ PNG ⇄ WebP
相互変換に対応

📄

PDF を画像に変換

PDF → PNG / WebP
各ページを個別の画像ファイルとして書き出し

🗂

画像を PDF に結合

JPG / PNG → PDF
複数の画像ファイルを 1 つの PDF に結合

📝

テキストを PDF に変換

TXT → PDF
日本語対応（MS ゴシック / CID フォント）

主な機能

🪟

モダンな GUI

CustomTkinter によるダーク / ライトモード対応のデスクトップアプリ

📁

ドラッグ＆ドロップ

ウィンドウ全体にファイルをドロップして即追加。ボタン選択も可能

✏️

出力ファイル名の入力

ファイル追加時に自動入力。複数ファイルは連番（_001, _002…）で保存

📊

プログレスバー

変換の進捗をリアルタイム表示。完了時にポップアップで通知

📦

exe ファイルに変換可能

PyInstaller で 1 つの exe にパッケージ化。Python 不要で配布できる

🇯🇵

日本語 PDF 対応

MS ゴシック / 游ゴシック / メイリオを自動検出して PDF に埋め込み

セットアップ

1

リポジトリをクローン

または ZIP でダウンロードして解凍してください。

git clone https://github.com/yumedebug/image-pdf-text-converter.git
cd image-pdf-text-converter
2

仮想環境を作成して有効化

python -m venv venv
venv\Scripts\activate   # Windows
source venv/bin/activate # macOS / Linux
3

依存ライブラリをインストール

pip install -r requirements.txt
4

アプリを起動

python converter.py
5

exe ファイルにパッケージ化（任意）

Windows 向けに 1 ファイルの exe を生成します。

pip install pyinstaller
pyinstaller converter.spec
